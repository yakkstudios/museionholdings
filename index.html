The Order · Gentlemen of Muse

Reputation-weighted, not capital-weighted.

Thirteen founding seats; 8 currently held. A bond posted per seat. A constitution of seven non-negotiable codes. The Order will adjudicate Veritas disputes, holds a 1.5× governance weight on Museion proposals, and stewards the cultural integrity of the whole house.

Code 01
Truth before tribe
When evidence and loyalty conflict, evidence wins. Seat-holders confront their own.
Code 02
No insider trades
No trading in any token, equity, or asset that intersects an open Veritas claim.
Code 03
No secret entanglements
Every external compensation, advisory seat, and material gift is disclosed annually.
Code 04
Confront in private
Disagreement is named within the Order before it is named outside the Order.
Code 05
No naming children
Investigations stop at adults. Family members of subjects are not named.
Code 06
Defend the indefensible
If the record clears the accused, the Order says so publicly. Even when it costs.
Code 07
改善 · improve the work
Every quarter, the Order re-reads its own record and amends what it has learned.
Membership is earned, not bought. The seat is not for sale.
